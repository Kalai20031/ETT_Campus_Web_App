# 🎓 ETT Campus Web Application

ETT Campus is a web-based educational institute platform developed using HTML, CSS, JavaScript, PHP, and MySQL.  
It provides a complete system for managing students, courses, tutorials, and registrations through a structured and user-friendly interface.



## 🚀 Features

- 🏠 Landing page for introduction and navigation  
- 🔍 Browse page to explore courses/content  
- 📘 Subject management system  
- 🎥 Tutorials page for learning materials  
- 📝 Student registration system  
- 🔐 Login and signup authentication system  
- 📄 Model page for structured content display  
- 👨‍🎓 User-friendly institute management interface  



## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  
- PHP  
- MySQL  
- Apache / XAMPP Server  



## 🗄️ System Architecture

- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  
- Server Domain Concept: `exam`  
- Data Resource: ETT Campus database (multi-server setup concept)


## ⚙️ Installation & Setup

1. Clone the repository:

git clone https://github.com/your-username/ett-campus.git


2. Move project to server directory:
- XAMPP → `htdocs/`

3. Import database:
- Open phpMyAdmin  
- Create database (e.g., `ett_campus`)  
- Import `.sql` file  

4. Run the project:

http://localhost/ett-campus/
